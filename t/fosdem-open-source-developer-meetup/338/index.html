FOSDEM Open Source Developer Meetup

Boris

FOSDEM is a community-led event for Open Source developers and enablers. It has a lot of working rooms that go deep into specific programming languages, licencing, community management etcetera. Bringing together over 5000 passionate nerds to strengthen the social bonds that make up the Open Source ecosystem.

FOSDEM takes place at the ULB Solbosch Campus, Brussels, Belgium, Europe, Earth.

Entry is free and no ticket is necessary.

https://fosdem.org/
